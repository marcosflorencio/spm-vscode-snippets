# spm-vscode-snippets
Snippets from spm project

Options

- [x] Controller - Controller with config
- [x] IIFE - Simple IIFE
- [x] func-injec - Function and $inject


## How to use? ##

1. Inside Visual Studio Code press
   ```bash
   CTRL + P
   ```
   
2. Type the following in the console window
  ``` bash
  ext install spmSnippets
  ```
3. Hit enter.
